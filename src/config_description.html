Config.json Properties and Valid Values

"button_position"
Values Description
"outside" (default) Buttons are to the left and right of the element
"overlay" Buttons are laid over the element.
"transition_time"
Value Description
number, time in milliseconds (default: 10) Total length of time it takes to move to target
"number_of_frames_per_transition"
Value Description
number, integer (default: 40) number of frames between one element and the next
