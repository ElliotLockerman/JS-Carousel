Config.json Properties and Valid Values

Display
Attribute Values Description
"button_position" "outside" (default) Buttons are to the left and right of the element
"overlay" Buttons are laid over the element.
Animation
Attribute Value Description
"transition_time" number, time in milliseconds (default: 10) Total length of time it takes to move to target
"number_of_frames_per_transition" number, integer (default: 40) number of frames between one element and the next
